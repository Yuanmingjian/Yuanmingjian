验证二叉搜索树

Created2022-11-12|Updated2023-09-23
|Post View:

解法一

运用递归 因为中序遍历的顺序就是二叉搜索树的值从小到大的顺序，利用这一点就可以判断 需要指定一个pre来存上一个节点

if (prev != null && prev.val >= root.val)
return false;
prev = root;

每一次递归都在执行这两步 这两步不能调换位置

###解法二

运用栈 一次性存两个数据进入栈中，利用先进后出的特点，可以先得到最左边的子节点，与上一个中间的结点比较，再把pre赋于左节点，然后这一个中间的节点在于pre比较，依次类推（较难理解）
